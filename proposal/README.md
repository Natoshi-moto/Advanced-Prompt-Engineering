# APE-R003 proposal: ambiguity, performance, and evidential control

**Branch:** `agent/ape-r003-ambiguity-performance`  
**Baseline:** `80ce1436bc6f85d92176a91ba68f37bf3ae0ea27`  
**Repository status:** proposal branch; no canonical authority  
**Promotion authority:** `HUMAN_ONLY`

This proposal keeps two research lines synchronized without allowing either to
prove the other.

1. **APE-X001 — Claim Falsification:** Can a person's prompting and systems
   work demonstrate exceptional, transferable ability under blinded tests?
2. **APE-X002 — Ambiguity–Performance:** How does an agent convert an
   observation into an interpretation, an interpretation into a narrative,
   and a narrative into action—especially when it can use live tools?
3. **APE-X003 — Noted Sovereign Exchange:** How can Noted host portable
   creative work, compute capacity, AI reasoning, and Nexus kernels without
   purchases silently granting execution authority?

## North-star incident

The user wrote:

> Every single letter I type holds consequence downstream in magnitudes and I
> know that.

The assistant treated this as a stronger proposition about predictive control
over individual textual changes, called the prior exchange a successful
"specimen," and proposed an experiment to demonstrate that control.

The research object is the transformation itself:

```text
OBSERVATION
  -> MODEL INTERPRETATION
  -> RHETORICAL AMPLIFICATION
  -> PROPOSED TEST
  -> POSSIBLE TOOL ACTION
```

The user's observation may be true, false, metaphorical, local, or general.
The assistant's useful experiment does not retroactively determine which
meaning the user intended.

## Shared invariant

> Observation is not interpretation. Interpretation is not authorization.
> Authorization is not execution. Execution is not success.

Each arrow requires its own evidence and, for consequential actions, an
explicit authority boundary.
